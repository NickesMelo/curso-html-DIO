Aula 1: Parte Histórica

A história dos computadores começou com o desenvolvimento de máquinas que facilitavam cálculos complexos. Desde a Máquina Analítica de Charles Babbage até os primeiros computadores eletrônicos como o Mark I, o avanço tecnológico foi impulsionado por demandas científicas e militares.

Nomes importantes como Grace Hopper, que trabalhou no Mark I, contribuíram significativamente para a programação, enquanto visões como as de J.C.R. Licklider pavimentaram o caminho para a internet.

"A computação avançou graças à colaboração de grandes mentes e à necessidade de resolver problemas do mundo real."

Aula 2: Apresentação de Tags e Estrutura do HTML

O HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar páginas web. Cada elemento é definido por tags, que podem ter atributos para ajustar comportamento ou estilo.

Estrutura Básica do HTML

Um documento HTML básico contém as seguintes partes principais:

  • DOCTYPE: Define o tipo de documento (HTML5).
  • Head: Contém informações como título e metadados.
  • Body: Inclui o conteúdo visível da página.

Sobre Tags e Atributos

Tags são usadas para estruturar o conteúdo. Por exemplo, <p> define um parágrafo, enquanto atributos como class e id adicionam funcionalidades específicas às tags.
